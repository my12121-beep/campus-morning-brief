## GitHub / ClawHub release post

Campus Morning Brief / 校园晨报 `v1.0.3` is available.

This is a reusable AI-agent skill for creating concise, personalized, source-backed morning briefs for students. It combines local weather, age-appropriate news, interest-based updates, study guidance, reminders, and a concrete action prompt into a brief that can be read in three to five minutes.

Release highlights:

- Bilingual documentation and workflow guidance.
- Structured JSON validation.
- Markdown and HTML rendering.
- Student-safe content guidelines.
- GitHub contribution, issue, and pull request templates.
- A reusable promotion kit for sharing the project.

# Content Guidelines / 内容规范

## Required sections / 必备栏目

1. Weather: city, conditions, temperature range, rain probability when available, clothing advice, and one campus-safety note. / 天气：城市、天气状况、温度范围、可用时的降水概率、穿衣建议及一条校园安全提示。
2. Quote: a short quotation in the primary language plus an English version and author. Use public-domain wording or a short paraphrase when attribution is uncertain. / 每日一句：主要语言短句、英文版本及作者；出处不确定时使用公共领域文本或简短意译。
3. General news: default to four items spanning more than one subject or region. / 综合要闻：默认四条，覆盖多个主题或地区。
4. Interest news: default to three items aligned with configured interests without becoming advertising. / 兴趣拓展：默认三条，贴合配置兴趣但不得变成广告。
5. Study guidance: one technique applicable today. / 学习方法：提供一项当天即可实践的方法。
6. Action anchor: one small, observable first action. / 行动锚点：提供一个微小且可观察的起始动作。
7. Reminders: include only user-provided or configured reminders. / 今日提醒：仅包含用户提供或已配置的提醒。

## Age calibration / 年龄适配

- `primary-school`: Use short sentences, explain uncommon terms, and avoid distressing events unless directly relevant to safety. / 使用短句并解释生词；除非与安全直接相关，否则避免令人不安的事件。
- `middle-school`: Add concise context and explain relevance without simplistic moral conclusions. / 补充简明背景及相关性，避免简单化的道德结论。
- `high-school`: Allow more policy, science, and economic context while defining specialist terms. / 可增加政策、科学和经济背景，同时解释专业术语。
- `college`: Use an adult reading level while retaining the short morning format. / 采用成人阅读水平，但保持晨报简洁。

## News selection / 新闻选择

- Prefer official agencies, original research, recognized newsrooms, and direct announcements. / 优先采用官方机构、原始研究、可信新闻机构及直接公告。
- Include publisher, publication date, and direct URL. / 必须包含发布机构、发布日期和直达链接。
- Avoid multiple items about the same event. / 避免多条内容重复报道同一事件。
- Label analysis, forecasts, preliminary research, and unverified claims accurately. / 准确标注分析、预测、初步研究及未经证实的说法。
- Do not infer facts from headlines alone. / 不得仅根据标题推断事实。
- Keep each summary to one or two sentences and explain relevance without fear-based framing. / 每条摘要控制在一到两句，说明相关性但避免制造恐慌。

## Writing / 写作要求

- Use the profile language consistently except for the bilingual quote. / 除双语名言外，全文使用配置中的主要语言。
- Do not reveal unnecessary private profile fields. / 不披露无关的私人配置字段。
- Do not diagnose health conditions or provide individualized medical, legal, or financial advice. / 不进行健康诊断，也不提供个性化医疗、法律或财务建议。
- Make weather advice conditional when the forecast is uncertain. / 天气预报不确定时，建议应采用条件式表达。
- End with a specific action instead of generic encouragement. / 以具体行动结束，而不是泛泛鼓励。
